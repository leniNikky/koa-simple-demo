# koa-simple-demo
koa2练习demo
## Installing
```
npm install
node index.js
```
在浏览器输入127.0.0.1:3000访问了

## 各demo文件对应内容说明
demo1: async和await的使用.
demo2: get请求数据的接收.
demo3: Post请求如何接收
demo4: Koa-bodyparser中间件.
demo5: Koa2原生路由实现.
demo6: Koa-router中间件（1）.
demo7: Koa-router中间件 (2) 层级.
demo8: Koa-router中间件 (3) 参数.
demo9: cookie的使用.
demo10: ejs模板.
demo11: koa-static静态资源中间件.
